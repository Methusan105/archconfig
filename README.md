# 🐧 Arch Linux Configuration

A simple set of scripts to automate the installation and post-install setup of **Arch Linux**.

This repository contains:

- ⚙️ An **Archinstall configuration generator**
- 🚀 A **post-install setup script**
- 🎨 MacTahoe KDE themes
- 💾 ZRAM configuration
- 🧠 Automatic RAM cache flushing
- 🔊 PipeWire audio setup
- 🌐 Flatpak + Flathub configuration
- 📦 Brave, Visual Studio Code and Stremio installation
- ⚡ Intel undervolt service
- 🖥️ GRUB configuration with OS Prober enabled

---

# 📋 Requirements

- UEFI system
- Internet connection
- Latest Arch Linux ISO
- Root/sudo access

---

# 🚀 Installation

## 1. Boot the Arch Linux ISO

At the live ISO prompt, load the Norwegian keyboard layout:

```bash
loadkeys no
```

---

## 2. Download the Archinstall configuration script

```bash
curl -O https://raw.githubusercontent.com/Methusan105/archconfig/main/archconf.sh
```

Make it executable:

```bash
chmod +x archconf.sh
```

Run it:

```bash
./archconf.sh
```

The script will generate an Archinstall configuration that can be imported directly into the Archinstall installer.

---

# 💿 Install Arch Linux

Launch the installer:

```bash
archinstall
```

When prompted for a configuration file, select the JSON file generated by `archconf.sh`.

Complete the installation and reboot into your new system.

---

# 🔧 Post-install Setup

After the first boot into Arch Linux, run:

```bash
curl -fsSL https://raw.githubusercontent.com/Methusan105/archconfig/refs/heads/main/setup.sh | sudo bash
```

The setup script will automatically configure your system.
